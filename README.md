# Proof of Concept .NET App

A demonstration project showcasing modern .NET technologies.

## Features

- **Blazor**: Interactive web UI with C#.
- **ASP.NET (with Controllers)**: RESTful API endpoints.
- **EFCore**: Database access via Entity Framework Core.
- **MSSQL Server**: SQL Server as the database backend.
- **Automapper**: Object-to-object mapping.
- **FluentValidation**: Robust model validation.

## Getting Started

1. **Clone the repository**
2. **Configure MSSQL connection string**
3. **Run migrations**
4. **Start the application**
